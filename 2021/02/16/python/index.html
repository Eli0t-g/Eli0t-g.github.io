寒假学习python是我没想到的，但是的确学到了不少知识。在这里分享一下我所学到的与我总结的关于python的知识

编译器

我认为目前有两个编译器比较好，一个是pycharm，另一个是VScode。这两个各有优劣，pycharm专业版比较正规，功能齐全，对初学者比较友好，但不支持其它编程语言。VScode支持非常多的插件，是当今主流编译器，而且十分简洁，还可以做成自己喜欢的风格，支持多语言，同时还非常小，但是功能却不如pycharm齐全。所以如果是主学python我推荐使用python，如果还学其它语言我推荐VScode。

函数

记录一些我容易忘记的，便于以后复习使用

format() 函数

作用：格式化字符串

用法：str.format()

实例：

  • >>>print("{} {}".format("hello", "world")")   # 不设置指定位置
'hello world'

  • >>>print('Hello {w}'.format(w='World')) # 指定参数
Hello World

输出：

0 a

1 b

2 c

ord() 函数

作用：返回指定对象的ASCII码对应的十进制

用法：ord(object)

实例：

>>>ord(‘a’)

97
